# Images par défaut des plats

Remplacez ces fichiers par vos propres photos (JPG/WebP, ~800×500 px, < 150 Ko).

| Fichier | Usage |
|---------|--------|
| `couscous.jpg` | Couscous |
| `tajine.jpg` | Tajine |
| `salade.jpg` | Salades |
| `grillades.jpg` | Brochettes, grillades |
| `dessert.jpg` | Desserts, pâtisseries |
| `boisson.jpg` | Boissons |
| `plat-generic.jpg` | Catégorie inconnue |

La correspondance est définie dans `src/lib/menuDefaultImages.js` (nom du plat + catégorie).
